# 青蝠

> 手持铁叉枪，身着布衣裳。
> 林深天地白，正是好猎场。

昔年，禅光村有个姓王的刀手，他幼时失怙，母亲养不活他，就送他去山下屠户家当徒工，让他习得了一身好本事。

这日，刀手去富户家杀猪，主人家多赏了他一条猪腿。刀手想将其送给母亲享用，就连夜赶路回村。行至半路，见山道旁有人，手持一杆猎叉，着一袭旧布衫，招呼道：“阿哥，我是村里的猎户，在这坡下摽兔。夜里有些怕鬼，想与你同行。”

刀手点头答应，二人继续赶路。没多久，猎户道：“听闻近日山道上闹妖怪哩。”

刀手笑道：“妖怪有什么好怕的？若是遇着，我就用猪腿抡他，用屠刀砍他。”

又走一阵，猎户道：“阿哥既不怕妖怪，想必很有本事？”

刀手笑道：“我自小宰杀畜生，一刀就能直中要害，一劈就能断骨断筋。”

猎户生气道：“妖怪怎能和畜生并论？”

刀手严肃道：“在我看来，世上没有妖怪。替天行道的，都是豪杰；欺负良善的，都是猪狗。”

村子遥遥在望，猎户又道：“你既不怕我，何不转身看看？”

刀手早有所料，抡起猪腿就将其打倒在地，举刀就砍。

那怪早已走了气势，急忙扇开翅膀飞到半空，啐道：“遇到个硬茬，晦气晦气！”

骂完立刻逃走了。

咦，都说柿子要拣软的捏，做人还是刚强些好，硬气起来，妖怪都怕你几分哩。
